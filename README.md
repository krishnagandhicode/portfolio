# Krishna Gandhi - Portfolio Website

This repository contains my production portfolio website, built to present my projects, technical strengths, and practical engineering experience in a clear, recruiter-friendly format.

## Live Site

- https://krishnagandhi.me

## Overview

- Responsive portfolio interface with polished motion and 3D visuals
- Project showcase with live deployments
- Experience and skills sections tailored for professional review
- Netlify deployment with GitHub-based auto-deploys

## Tech Stack

- Frontend: React, Vite
- Animation and Interaction: GSAP, CSS
- 3D UI: Three.js
- Deployment: Netlify
