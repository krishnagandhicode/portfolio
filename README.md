# Krishna Gandhi - Portfolio Website

This repository contains my personal portfolio website, built to showcase selected projects, technical skills, and development experience.

## Live Website

- https://krishnagandhi.me


## Tech Stack

- Frontend: React, Vite
- Styling and Animation: CSS, GSAP
- 3D and UI: Three.js
- Deployment: Netlify (GitHub auto-deploy)

## Highlights

- Responsive, modern portfolio interface
- Scroll-based motion and interactive UI effects
- 3D hero and visual components
- Curated project showcase with live previews
- Experience and tech stack sections with custom content

## Learning Focus

- Debugging production-ready frontend issues
- Handling deployment edge cases on Linux-based CI
- Building maintainable React component structure
- Optimizing animation and rendering performance
